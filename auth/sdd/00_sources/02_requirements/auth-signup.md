# 회원가입 OTP: 요구사항 원문 (강의 데모용 가상)

> 실재 개인정보·실명 없음. 이메일은 불투명 예시값(`a@x.com`)만 사용.

- 회원가입은 이메일 OTP(6자리) 검증을 통과해야 완료된다.
- OTP는 발급 후 5분(300초) 내에만 유효하다.
- 인증번호를 5회 틀리면 해당 OTP를 잠근다(무차별 대입 방지).
- 같은 사용자의 가입 요청이 재실행돼도 계정이 중복 생성되지 않는다.
- OTP 입력 화면(signup_otp)은 디자인 스냅샷과 일치해야 한다.
- 기존 로그인 흐름은 회원가입 추가로 깨지지 않는다.
